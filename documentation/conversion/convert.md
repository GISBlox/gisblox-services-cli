# Conversion
Contains commands to convert WKT geometries into GeoJson.

Available commands:
- [convert](#gbs-convert)

## gbs convert
Converts WKT geometries into GeoJson.

### Usage
```
gbs convert [options]
```

### Options
```
--help          Show help information.
-w|--wkt <wkt>  The WKT geometry to convert.

```
